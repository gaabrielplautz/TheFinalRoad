![alt_text](https://github.com/gaabrielplautz/TheFinalRoad/blob/83b89a2393aa49ba5d31ab532ab289765477205c/assets/IMG/Player.png). # THE FINAL ROAD
The Final Road is a 2D arcade game where you take control of a truck and have to dodge oncoming vehicles. As the game progresses, the speed of the NPCs increases, making it more difficult. Points are added for each overtaking maneuver, and the score is saved in a SQLite3 database.
